# 💻 Amritesh Kumar Rai: Full-Stack Developer Portfolio

This is the personal portfolio website for **Amritesh Kumar Rai**, a Full-Stack Developer specializing in the **MERN stack**. This single-page application (SPA) serves as a dynamic showcase of my technical skills, professional expertise, and major development projects.

## ✨ Key Features

  * **Responsive Layout:** Fully optimized for desktop, tablet, and mobile devices.
  * **Smooth Navigation:** Uses React refs and smooth scrolling to seamlessly navigate between the main sections.
  * **Dynamic Tagline:** Features a custom Typewriter effect on the home screen to display core skills.
  * **Theme Toggle:** Includes a persistent light/dark mode toggle using local storage for personalized viewing.
  * **Project Showcase:** Dedicated section featuring live demos, code repositories, and detailed one-liner descriptions of flagship projects.
  * **Direct Contact:** Integrated contact form (via Formspree) and direct links to social and professional profiles.
  * **CV Download:** A floating component provides instant access to my resume.

## ⚙️ Local Setup

To run this portfolio locally on your machine for development or review:

1.  **Clone the repository:**

    ```bash
    git clone <YOUR_GIT_URL>
    cd my-portfolio
    ```

2.  **Install Dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the Development Server (using Vite):**

    ```bash
    npm run dev
    # or
    yarn dev
    ```

The application will be available at `http://localhost:5173` (or the port specified by Vite).
